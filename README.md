## Requirements
- Node v8.15.x

### Installing Dependencies
From within the root directory


1. Install project dependencies
```javascript
npm install
```

2. To create a client bundle
```javascript
npm run react-dev
```

3. To start the server
```javascript
npm start
```

5. Go to `localhost:3000
